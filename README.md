# Telmah Portfolio

Modern React + Tailwind CSS + Vite portfolio built for free Netlify deployment.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Netlify

1. Push this folder to GitHub.
2. Go to Netlify.
3. New site from Git.
4. Build command: `npm run build`
5. Publish directory: `dist`

## Customize

- Update project links in `src/data/portfolio.js`
- Add your real resume as `public/Telmah-Tantoh-Resume.pdf`
- Replace LinkedIn URL in `src/App.jsx`
